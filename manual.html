Telperion

User Manual

1. Introduction

Telperion is a specialized tool for building and analyzing syntactic trees under the Role and Reference Grammar (RRG) model. The application allows you to manage the three fundamental projections of the clause in a single workspace: Constituent Projection, Operator Projection, and Information Structure.

Initial morphosyntactic expression: Every diagram starts from a segmented expression, typically corresponding to words or multiword blocks. These base items act as anchors for both operator layers and information-structure units.

2. Start Screen and Segmentation

Upon launching Telperion, you will be greeted by the welcome screen where you can generate a new diagram or load an existing one.

Create a new diagram

  1. Enter your full sentence in the text input field.
  2. IPA Characters: Click the IPA button to open the IPA picker. You can use it on the start screen and also later on the canvas while editing base items or node labels.
  3. Adjust segments: To group or split tokens, click the space between items in the word-flow area. This toggles the boundary at that position.
  4. Use Split All to separate all words individually, or Clear to merge them all into a single block.
  5. Click Generate Initial Expression to proceed to the workspace canvas.

Load an existing diagram

You can drag and drop a .tlp file directly into the right-hand box, or click Load .tlp File to browse your computer’s files.

3. The Workspace Canvas

The main interface is divided into three key areas:

  • Top Bar: global actions such as load, save, export, undo/redo, and new diagram.
  • Central Canvas: the diagram area where you create and edit the three projections. Floating zoom controls are located in the top right corner of this area.
  • Right Sidebar: context-sensitive tools; what you see depends on the selected node and active projection mode.

Navigation and selection

  • Pan / scroll: Use your trackpad or mouse wheel to scroll. If the diagram becomes wider than the window, horizontal scrolling becomes available.
  • Select: click a node or label to select it. Click on empty space to clear selection.
  • Edit a label: Click on any label to select it and start typing.

Editing the Base Expression

  • If you missed a word or segment after generating: You do not need to start over. Simply select the closest base item on the expression and use ← Insert Left or Insert Right → to insert the missing item in the correct position, then edit its text normally.
  • Swap neighboring base items: You can swap adjacent base items using ← Swap Left and Swap Right →. This is especially useful if you notice an ordering issue after generation and want to fix it without redoing segmentation.
  • IPA while editing: When you edit a base item directly on the canvas, an IPA button is available next to the text field, allowing you to insert IPA symbols at the cursor position.
  • Secondary Gloss Line: To add a standard linguistic gloss layer, click + Add secondary gloss line. This creates a row of borderless, editable boxes directly below each base item.
    • The system automatically extends connection lines for operators and information structure to make room for this layer.
    • To hide this layer and clear its content, click ✕ Remove secondary gloss line.

Editing labels and Text Formatting

Telperion makes it easy to assign and format labels across the Constituent, Operator, and Information Structure projections using a special markup system. A Text Formatting Guide is always visible in the sidebar when a node is selected.

*text* Italics **text** Bold %text% Small Caps _text_ or _text Subscript (e.g., V_1_ or V_1 renders as V1) __text__ or __text Superscript (e.g., N__1__ or N__1renders as N1)
  • Dropdowns and Free Typing: When you click a label to edit it, an input field appears with a chevron on the right side. Clicking this chevron toggles a dropdown list offering context-appropriate RRG labels. You can click to select from the list, or type freely to enter any custom text.
  • Toggle behavior: Markers act as toggles. For example, CORE_P_ - ARG will render as COREP - ARG.
  • Combining styles: You can combine styles by nesting markers. For example, *%pst%* renders as pst.
  • Press Enter to confirm a label edit.
  • Press Esc to cancel an edit or close an open dropdown menu.

Global Projection Controls

When you deselect all nodes by clicking on empty space, the sidebar displays the global projection controls:

  • Toggle Visibility: you can hide or show any of the three projections by clicking the checkbox next to each one. This is useful for focusing on specific layers while presenting or analyzing.
  • Dimming Projections: when both Operator Projection and Information Structure are active simultaneously, the visual space below the base expression can become crowded. Under Dim projections, you can select whether to dim the operators or the information structure lines.

4. Constituent Projection

The constituent projection represents the hierarchical structure of the clause. Click Constituent Projection in the sidebar to initialize the tree. Once started, this button turns into a status indicator, and you can build the tree by interacting directly with the nodes.

Building the tree

  • Daughter nodes: Select any node and use the Number selector control in the sidebar to add or subtract daughter nodes.
  • Add Parent: Select a node and click + Add Parent to wrap it under a new superior node.
  • Add Periphery: Select a node and use the Add Left → or ← Add Right buttons to attach a peripheral node to that structural level.
  • Delete nodes: Select the node in the canvas, then click the red ✕ Delete Node button in the right sidebar.
  • Spine alignment: for syntactic nodes, you can center the node vertically when the option is available in the sidebar.
  • Swapping nodes: you can swap nodes in the constituent tree using the ← Swap Left or Swap Right → buttons. Note: this action is only available before linking the node to the base items.

Node hover controls

Telperion provides small green quick-action buttons that appear when you hover over a node. These buttons are designed to speed up tree building without moving your cursor to the sidebar.

  • Add child (+): When you hover over a constituent node, a small green + appears centered on the bottom edge. Click it to create new daughter nodes under that node.
  • Visibility: The quick buttons are shown on hover.

Links to the morphosyntactic expression

To link your syntactic structure with the expression items:

  1. Select the terminal node.
  2. In the sidebar, under Linking to expression item, choose the corresponding item from the dropdown.
  3. Peripheries: a periphery may link to multiple expression items using checkboxes.
  4. Draw link as triangle: when a node dominates a full phrase, enable Draw link as triangle to replace multiple word-links with a single triangle over the relevant span.

Incoming Links

If you need to review or quickly remove links pointing to a specific base item from syntax or operators:

  1. Select any item in the base expression.
  2. Look at the Incoming Links section in the right sidebar.
  3. You will see a list of all syntactic nodes and operator nodes that are currently pointing to this item. Uncheck the boxes to unlink them.

5. Operator Projection

The operator projection is built downwards from the base expression. Click on any base item to begin adding operator layers.

Layer and operator creation

  • Add a layer: Select a base item and click + Add Operator Layer.
  • Add operators: Select the created layer and click Add operator → or ← Add operator.
  • Position adjustment: move layers vertically with ↑ Move Up and ↓ Move Down. Operators can be moved horizontally using ← Move Left or Move Right →.

Layer hover controls

Operator layers also include green hover buttons to accelerate common actions directly on the canvas.

  • Add child (+): When a layer node is hovered or selected, a green + appears on its bottom edge. Click it to add the next layer below.
  • Add operator on the left (→): A green button on the left edge adds an operator node to the left of the selected layer.
  • Add operator on the right (←): A green button on the right edge adds an operator node to the right of the selected layer.
  • Visibility: These buttons are visible on hover.

Cross-spine links

Telperion supports links between operator projections of different nuclei:

  1. Select the source layer.
  2. In the sidebar, click Link to secondary superior node.
  3. An on-screen indicator will appear. Click the target layer in another spine or a base item to generate the transversal link.
  4. Cancel linking mode: press Esc or click the linking button again.

6. Information Structure

To analyze how information is distributed, click on any base item to activate Information Structure options.

Information Units

  • Select a base item and click + Mark as Information Unit.
  • Span across multiple items: use Grow Left, Grow Right, Shrink Left, or Shrink Right.
  • Remove an IU: with the IU selected, click ✕ Remove Information Unit.

Focus domains and Speech Act

Telperion features an interactive selection mode for assigning focus domains:

  1. Make sure no specific node is selected, then click the Information Structure button to open the Focus Domains panel.
  2. Potential and Actual Focus: Click the selection button to enter selection mode, then click the target IU on the canvas. The PFD is drawn as a dashed triangle, and the AFD as a solid one.
  3. Special Focus Domains: Mark IU as Completive Focus or Contrastive Focus, then click the target IU on the canvas.
  4. Telperion automatically calculates triangle geometry and draws lines towards the Speech Act node.

Speech Act node position

Select the Speech Act label on the canvas to adjust its vertical position using ↑ Move Up and ↓ Move Down in the sidebar. Use Reset position to return to the default.

7. Saving and Exporting

  • Save: downloads a .tlp file. This is Telperion’s native, fully editable format.
  • What is a .tlp file?: It is simply a JSON file produced by Telperion. The extension is just a custom label. You can open it in any text editor, although you normally should not edit it manually.
  • Export PNG: generates a high-resolution image with clean rendering and appropriate padding.
  • Auto-save: Telperion periodically stores recent work in your browser to help recover from accidental tab closes. It is a safety net, not a replacement for saving a .tlp when you finish.

8. Keyboard Shortcuts

Telperion supports quick shortcuts to improve your workflow:

  • Ctrl / ⌘ + S : Save file (.tlp)
  • Ctrl / ⌘ + Z : Undo
  • Ctrl / ⌘ + Y or Ctrl / ⌘ + Shift + Z : Redo
  • Ctrl / ⌘ + + : Zoom In
  • Ctrl / ⌘ + - : Zoom Out
  • Ctrl / ⌘ + 0 : Reset Zoom to 100%
  • Enter : Confirm label edit
  • Esc : Cancel edit, exit modes, close dropdown
