Telperion

User Manual

1. Introduction

Telperion is a specialized tool for building and analyzing syntactic trees under the Role and Reference Grammar (RRG) model. The application allows you to manage the three fundamental projections of the clause in a single workspace: Constituent Projection, Operator Projection, and Information Structure.

Initial morphosyntactic expression: every diagram starts from a segmented expression (one or more “base items”, typically corresponding to words or multiword blocks). These base items act as anchors for both operator layers and information-structure units.

2. Start Screen and Segmentation

Upon launching Telperion, you will be greeted by the welcome screen where you can generate a new diagram or load an existing one.

Create a new diagram

  1. Enter your full sentence in the text input field.
  2. IPA Characters: Click the IPA button to open the IPA picker. You can use it on the start screen (sentence entry) and also later on the canvas while editing base items or node labels.
  3. Adjust segments: to group or split tokens, click the space between items in the word-flow area. This toggles the boundary at that position.
  4. Use Split All to separate all words individually, or Clear to merge them all into a single block.
  5. Click Generate Initial Expression to proceed to the workspace canvas.

Load an existing diagram

You can drag and drop a .tlp file directly into the right-hand box, or click Load .tlp File to browse your computer’s files.

3. The Workspace Canvas

The main interface is divided into three key areas:

  • Top Bar: global actions such as load, save, export, undo/redo, and new diagram.
  • Central Canvas: the diagram area where you create and edit the three projections. Floating zoom controls are located in the top right corner of this area.
  • Right Sidebar: context-sensitive tools; what you see depends on the selected node and active projection mode.

Navigation and selection

  • Pan / scroll: use your trackpad/mouse wheel to scroll. If the diagram becomes wider than the window, horizontal scrolling becomes available.
  • Select: click a node/label to select it. Click on empty space to clear selection.
  • Edit a label: Click on any label to select it and start typing.

Editing the Base Expression (Lexical Equator)

If you realize you missed a word or segment after generating the initial tree, you can add it directly on the canvas without starting over:

  • Select any base item (word) in the diagram's lexical equator.
  • In the sidebar, under Expression Items, click ← Insert Left or Insert Right → to add a new empty base node adjacent to your selection.
  • IPA while editing: When you edit a base item directly on the canvas, an IPA button is available next to the text field, allowing you to insert IPA symbols at the cursor position.

Global Projection Controls

When you deselect all nodes (by clicking on empty space), the sidebar displays the global projection controls:

  • Toggle Visibility: you can hide or show any of the three projections by clicking the checkbox next to each one. This is useful for focusing on specific layers while presenting or analyzing.
  • Dimming Projections: when both Operator Projection and Information Structure are active simultaneously, the visual space below the equator can become crowded. Under Dim projections, you can select whether to dim the operators or the information structure lines, making the active focus clearer.

4. Constituent Projection

The constituent projection represents the hierarchical structure of the clause. Activate Constituent Projection in the sidebar to begin working in this layer.

Building the tree

  • Daughter nodes: select any node and use the Number selector control in the sidebar to add or subtract daughter nodes.
  • Add Parent: select a node and click + Add Parent to wrap it under a new superior node.
  • Add Periphery: select a node and use the Add Left → or ← Add Right buttons to attach a peripheral node to that structural level.
  • Delete nodes: select the node in the canvas, then click the red ✕ Delete Node button in the right sidebar (this removes the node and its descendants).
  • Spine alignment: for syntactic nodes, you can center the node vertically (spine) when the option is available in the sidebar.
  • Swapping nodes: you can swap nodes in the constituent tree using the ← Swap Left or Swap Right → buttons. Note: this action is only available before linking the node to the base items.

Node hover controls (quick buttons)

In addition to the right sidebar controls, Telperion provides small green “quick action” buttons that appear when you hover over a node (and remain visible when the node is selected). These buttons are designed to speed up tree building without moving your cursor to the sidebar.

  • Add child (+): When you hover/select a constituent node, a small green + appears centered on the bottom edge. Click it to create new daughters nodes under that node.
  • Visibility: The quick buttons are shown on hover; they are always available while the node remains selected.
  • Scope: These hover buttons apply to nodes in the Constituent Projection.

Editing labels (Applicable to all projections)

Telperion makes it easy to assign and format labels across the Constituent, Operator, and Information Structure projections:

  • Dropdowns & Free Typing: When you click a label to edit it, an input field appears with a chevron (arrow) on the right side. Clicking this chevron toggles a dropdown list offering context-appropriate RRG labels. As you type, the list filters automatically. You can click to select from the list, or simply type freely to enter any custom text.
  • Subscripts: In free edition mode, you can automatically generate subscript formatting by using an underscore (_). For example, typing CORE_P will render the "P" as a subscript in the canvas.
  • Press Enter to confirm a label edit.
  • Press Esc to cancel an edit or close an open dropdown menu.

Links to the morphosyntactic expression

To link your syntactic structure with the expression items (base segments):

  1. Select the terminal node (the lowest node in a branch).
  2. In the sidebar, under Linking to expression item, choose the corresponding item from the dropdown.
  3. Peripheries: a periphery may link to multiple expression items using checkboxes.
  4. Draw link as triangle: when a node dominates a full phrase, enable Draw link as triangle to replace multiple word-links with a single triangle over the relevant span.

Incoming Links (Managing connections from the base)

If you need to review or quickly remove links pointing to a specific item from the syntax or operators:

  1. Select a base expression item (a word) in the lexical equator.
  2. Look at the Incoming Links section in the right sidebar.
  3. You will see a list of all syntactic nodes and operator nodes that are currently pointing to this item. Uncheck the boxes to instantly unlink them from their respective projection.

5. Operator Projection

The operator projection is built downwards from the morphosyntactic expression. Click on any base node to begin adding operator layers.

Layer and operator creation

  • Add a layer: select a base item and click + Add Operator Layer (e.g., NUC, CORE, CLAUSE).
  • Add operators: select the created layer and click Add operator → or ← Add operator (e.g., ASP, TNS, MOD).
  • Position adjustment: move layers vertically (↑ Move Up / ↓ Move Down) to create space. Operators can be moved horizontally using ← Move Left or Move Right → to make the diagram clearer and avoid overlapping lines.

Layer hover controls (quick buttons)

Operator layers also include green hover buttons to accelerate common actions directly on the canvas.

  • Add child (+): When a layer node is hovered/selected, a green + appears on its bottom edge. Click it to add the next (single) layer below that stratum.
  • Add operator on the left (→): A green button on the left edge adds an operator node to the left of the selected stratum.
  • Add operator on the right (←): A green button on the right edge adds an operator node to the right of the selected stratum.
  • Visibility: These buttons are visible on hover and remain available while the stratum is selected.

Cross-spine links

Telperion supports links between operator projections of different nuclei (useful in juncture/nexus configurations):

  1. Select the source layer.
  2. In the sidebar, click Link to secondary superior node.
  3. An on-screen indicator will appear. Click the target layer in another spine or a base node to generate the transversal link.
  4. Cancel linking mode: press Esc (or click the linking button again) to exit without creating a link.

6. Information Structure

To analyze how information is distributed, click on any base node to activate Information Structure options.

Information Units (IU)

  • Select a base item and click + Mark as Information Unit.
  • Span across multiple items: use Grow Left/Right or Shrink Left/Right. If the IU covers more than one item, Telperion draws a bounding bracket automatically.
  • Remove an IU: with the IU selected, click ✕ Remove Information Unit.

Focus domains and Speech Act

Telperion features an interactive selection mode for assigning focus domains without cluttering the interface:

  1. Make sure no specific node is selected on the canvas to view the global Information Structure options in the sidebar.
  2. Potential & Actual Focus: Click the Select IU(s) as part of the PFD or Select IU(s) as part of the AFD button to enter selection mode, then click the target Information Unit(s) directly on the canvas. The PFD is drawn as a dashed triangle, and the AFD as a solid one.
  3. Special Focus Domains: For specific focus types, click Mark IU as Completive Focus (horizontal hatching) or Mark IU as Contrastive Focus (vertical hatching), then click the target IU on the canvas.
  4. Telperion automatically calculates the geometry of the triangles to prevent visual overlaps and draws lines towards the virtual Speech Act node.

Speech Act node position

Select the Speech Act label on the canvas to adjust its vertical position using ↑ Move Up / ↓ Move Down in the sidebar. Use Reset position to return to the default.

7. Saving and Exporting

  • Save: downloads a .tlp file. This is Telperion’s native, fully editable format. Use it to continue work later or move diagrams between computers.
  • Export PNG: generates a high-resolution image with clean rendering (selection outlines and UI-only controls are hidden) and appropriate padding. Useful for papers and slides.
  • Auto-save: Telperion periodically stores recent work in your browser to help recover from accidental tab closes. It is a safety net, not a replacement for saving a .tlp when you finish.

8. Keyboard Shortcuts

Telperion supports quick shortcuts to improve your workflow:

  • Ctrl / ⌘ + S : Save file (.tlp)
  • Ctrl / ⌘ + Z : Undo
  • Ctrl / ⌘ + Y or Ctrl / ⌘ + Shift + Z : Redo
  • Ctrl / ⌘ + + : Zoom In
  • Ctrl / ⌘ + - : Zoom Out
  • Ctrl / ⌘ + 0 : Reset Zoom to 100%
  • Enter : Confirm label edit
  • Esc : Cancel edit / exit modes / close dropdown
